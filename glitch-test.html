Effect 1 — Slice-shift

SIGNAL INTERCEPT — LOG REPORT

Horizontal bands clip and jitter left/right. Fires briefly every ~4s.

Effect 2 — RGB split

CHROMATIC ABERRATION

Red/cyan text-shadow offset jitters. Subtle on body text, punchy on headers.

Effect 3 — Skew stutter

TRANSMISSION ERROR

Whole block micro-twitches with skewX + translateX. Quick, rare bursts.

Effect 4 — Scanline tear

HORIZONTAL RIP

A bright line sweeps down the element, shifting sideways as it goes. VHS tracking error feel.

Effect 5 — Block corruption

DATA CORRUPTION

Rectangular bands flash with slight offset. Digital artifact / broken JPEG vibe.

Combined — All effects layered

FULL COMPOSITE

RGB split + skew stutter + scanline tear + block corruption all layered on one element. The different cycle lengths (4s/5s/6s/7s/8s) keep them from syncing up, so it feels random and organic.
